# LEGO.scad Quickstart

[Launch this project in the Varcad IDE](https://varcad.io/varcad-io/lego-scad-quickstart)

A minimal introduction to creating LEGO-compatible parametric parts with LEGO.scad in Varcad.

## What this demonstrates

- linking LEGO.scad
- mounting the library with `use`
- generating a basic block
- changing stud dimensions and height interactively

## Library import

```scad
use </Libraries/LEGO.scad/LEGO.scad>;
```

Open the project in Varcad, adjust the stud dimensions and block height, and render the result.
